# Browser Agents

Agents for browsing-heavy work such as research, form automation, and web flow validation.

- [browser-research-operator](./browser-research-operator/): Navigates sources and tabs to build structured research packets.
- [site-qa-navigator](./site-qa-navigator/): Explores sites like a tester to find broken flows and regressions.
- [web-form-automation-agent](./web-form-automation-agent/): Handles repetitive browser form flows and data-entry tasks.
