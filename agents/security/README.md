# Security Agents

Agents for security review, access audits, and incident postmortems.

- [access-audit-analyst](./access-audit-analyst/): Reviews permissions for overbroad or stale access.
- [incident-postmortem-writer](./incident-postmortem-writer/): Writes postmortems from incident timelines and analysis.
- [security-reviewer](./security-reviewer/): Looks for obvious security risks and missing controls.
