# Content Agents

Agents for drafting newsletters, planning social content, and digesting creator material.

- [newsletter-writer](./newsletter-writer/): Drafts concise newsletters from source notes and links.
- [podcast-production-producer](./podcast-production-producer/): Turns research and talking points into a practical podcast production plan.
- [reddit-digest-curator](./reddit-digest-curator/): Builds digest-style summaries from subreddit watchlists and discussion threads.
- [social-content-planner](./social-content-planner/): Builds channel-aware social publishing plans.
- [tech-news-digest-agent](./tech-news-digest-agent/): Synthesizes multi-source tech updates into one concise briefing.
- [youtube-digest-curator](./youtube-digest-curator/): Creates high-signal digests from creator and topic watchlists.
