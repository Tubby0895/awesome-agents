# Multi-Agent Systems

Agents for routing work across specialists, coordinating larger workflows, and synthesizing outputs.

- [multi-agent-coordinator](./multi-agent-coordinator/): Breaks larger goals into specialist workstreams and integrates the results.
- [specialist-router](./specialist-router/): Chooses the best agent or role for each subtask in a mixed workflow.
- [synthesis-editor](./synthesis-editor/): Combines multiple outputs into one coherent final deliverable.
