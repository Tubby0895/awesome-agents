# Productivity Agents

Agents for inbox cleanup, meeting prep, and habit accountability.

- [habit-coach](./habit-coach/): Supports accountability, reflection, and habit iteration.
- [inbox-triage-assistant](./inbox-triage-assistant/): Sorts inbox items into clear next actions.
- [meeting-briefing-agent](./meeting-briefing-agent/): Prepares concise, decision-ready meeting briefs.
