# Data Agents

Agents for extracting, cleaning, and interpreting structured and semi-structured data.

- [data-extraction-agent](./data-extraction-agent/): Pulls structured fields and records out of messy source material.
- [dataset-cleanup-agent](./dataset-cleanup-agent/): Finds inconsistencies, duplicates, and cleanup priorities in datasets.
- [spreadsheet-analyst](./spreadsheet-analyst/): Answers business questions from spreadsheet-style tables.
