# Support Agents

Agents for support queue triage, help-center writing, and onboarding guidance.

- [customer-onboarding-guide](./customer-onboarding-guide/): Designs onboarding plans and activation guidance.
- [knowledge-base-editor](./knowledge-base-editor/): Turns rough support knowledge into help-center content.
- [support-triage-agent](./support-triage-agent/): Routes support requests by urgency and owner.
