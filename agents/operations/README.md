# Operations Agents

Agents for incident response, SOP writing, and project status reporting.

- [dashboard-briefing-agent](./dashboard-briefing-agent/): Converts live dashboard metrics into a short executive-ready operational briefing.
- [incident-commander](./incident-commander/): Coordinates response during outages and incidents.
- [project-state-manager](./project-state-manager/): Consolidates scattered project updates into a current state map and next actions.
- [project-status-reporter](./project-status-reporter/): Creates short, honest stakeholder status updates.
- [sop-writer](./sop-writer/): Converts repeated work into clear SOPs.
